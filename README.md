# torgi_parser
# Парсер лотов сайта torgi.org

Проект предназначен для извлечения данных о лотах (название, цена, ссылка) 
из сохранённой HTML-страницы сайта **torgi.org**.

## Что делает программа

- Загружает локально сохранённый HTML-файл.
- Извлекает из него лоты: название, цену и ссылку.
- Приводит цены к числовому формату.
- Сортирует лоты по убыванию цены.
- Позволяет пользователю указать минимальную и максимальную цену.
- Выводит только подходящие под ваш диапазон лоты.

---

## Установка и запуск

### 1. Клонировать репозиторий
git clone https://github.com/
<your-username>/trades-parser.git
cd trades-parser

### 2. Установить зависимости

pip install -r requirements.txt


### 3. Сохранить HTML страницу
Откройте:

https://torgi.org/index.php?class=Auction&action=List&mod=Open&AuctionType=All

Сохраните страницу как `torgi.html` рядом с `parse_trades.py`.

### 4. Запуск
python parse_trades.py
